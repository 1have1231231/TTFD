# Инструкция по настройке статистики на сайте

## Что сделано

Discord бот теперь запускает HTTP API на порту 8080, который отдаёт статистику сервера.
Сайт получает данные с этого API каждые 30 секунд.

## Настройка на Railway

### Шаг 1: Discord Bot

1. Открой проект Discord бота на Railway
2. Перейди в **Settings → Networking**
3. Включи **Generate Domain** (если ещё не включено)
4. Скопируй URL (например: `ttfd-production.up.railway.app`)
5. Перейди в **Variables**
6. Добавь переменную:
   - **Name**: `STATS_API_PORT`
   - **Value**: `8080`

### Шаг 2: Website

1. Открой проект сайта на Railway
2. Перейди в **Variables**
3. Добавь переменную:
   - **Name**: `DISCORD_BOT_API_URL`
   - **Value**: `https://ttfd-production.up.railway.app` (URL из шага 1)

### Шаг 3: Деплой

1. Закоммить все изменения:
   ```bash
   git add .
   git commit -m "Интеграция статистики через HTTP API"
   git push
   ```

2. Railway автоматически задеплоит оба сервиса

3. Подожди 2-3 минуты

### Шаг 4: Проверка

1. Открой сайт
2. Нажми F12 (консоль браузера)
3. Должны появиться логи:
   ```
   🔄 Загрузка статистики...
   📡 Ответ получен: 200
   📊 Данные: {online_members: 5, total_members: 41, ...}
   ✅ Статистика обновлена успешно
   ```

4. На сайте должны отображаться:
   - Количество онлайн участников
   - Общее количество участников
   - Топ-10 игроков по XP с аватарками

## Если что-то не работает

### Проблема: Сайт показывает 0 онлайн

**Решение:**
1. Проверь что Discord бот запущен на Railway
2. Проверь что переменная `DISCORD_BOT_API_URL` правильная
3. Открой в браузере: `https://your-bot-url.railway.app/api/stats`
   - Должен вернуть JSON с данными

### Проблема: Карточки игроков не загружаются

**Решение:**
1. Открой консоль браузера (F12)
2. Посмотри на ошибки
3. Проверь что API возвращает массив `top_players`

### Проблема: API не отвечает

**Решение:**
1. Открой логи Discord бота на Railway
2. Найди строку: `✅ Stats API запущен на порту 8080`
3. Если её нет - бот не запустил API
4. Проверь что в Variables есть `STATS_API_PORT=8080`
