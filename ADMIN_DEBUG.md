## Как проверить
1. Откройте сайт в браузере
2. Авторизуйтесь через Discord
3. Откройте консоль разработчика (F12)
4. Посмотрите на логи в консоли - должны быть сообщения типа:
   - `🔍 checkAdminAccess: Checking admin rights for user: 739566637400981506`
   - `📡 Admin check response status: 200`
   - `📊 Admin check response data: {is_admin: true, user_id: "739566637400981506", ...}`

## Если не работает
Проверьте в консоли браузера ответ от `/api/admin/check`:
- `user_id` должен быть `"739566637400981506"`
- `admin_id` должен быть `"739566637400981506"`
- `is_admin` должен быть `true`
